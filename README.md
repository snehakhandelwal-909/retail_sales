# 📊 Sales Performance Dashboard — Excel

A end-to-end Business Intelligence project built entirely in Microsoft Excel, analyzing 500+ sales transactions across regions, products, and salespersons for the year 2023.

---

## 📁 File Structure

| Sheet | Description |
|-------|-------------|
| `Raw Data` | 500+ transaction records with calculated columns |
| `Region Summary` | Revenue aggregated by region |
| `Monthly Summary` | Revenue trend across Jan–Dec 2023 |
| `Product Summary` | Revenue breakdown by product |
| `Analysis` | Salesperson performance table + Product × Region matrix |
| `Dashboard` | Visual KPI dashboard with charts |

---

## 🔧 Excel Skills Demonstrated

- **Formulas:** `SUMIF`, `COUNTIF`, `AVERAGEIF`, `SUMIFS`, nested `IF`, `IFERROR`
- **Cross-sheet references:** All summaries and KPIs pull live from Raw Data
- **Conditional Formatting:** Color scales, data bars, rule-based cell highlighting
- **Charts:** Bar chart, Pie chart, Line chart — all linked to source data
- **Data tools:** Auto-filter, frozen panes, structured table layout
- **Data model:** Multi-sheet architecture (raw → summaries → dashboard)

---

## 🗂️ Dataset

- **Rows:** 500 sales transactions
- **Columns:** Date, Region, Product, Salesperson, Units Sold, Unit Price, Revenue, Profit, Profit Margin %, Revenue Category
- **Period:** January – December 2023

---

## 🚀 How to Use

1. Download `Sales_Dashboard_v2.xlsx`
2. Open in Microsoft Excel (2016 or later recommended)
3. Navigate to the **Dashboard** tab for the visual overview
4. Use the **Analysis** tab to explore salesperson and product-region breakdowns
5. All sheets update automatically if data in `Raw Data` is changed
